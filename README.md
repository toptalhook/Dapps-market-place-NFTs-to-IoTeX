## Available Wallet
- Torus ( if you not have metamask)
- Metamask ( Connect manually SET RPC NETWORK )
## Available NETWORK CHAIN
- IOTEX
- ETHEREUM
- Binance Smart Chain
- Polygon (Matic) 
## Support nfts
- ART
- MUSIC
- INTERACTIVE
- MEMES
- SPORTS
- MISC
- NFSW
- ANIME

## FEATURES
- SET AUCTION NFTs WITH DEADLINE
- NFT DETAIL SELLER OR BUYERS
- MULTI BATCH TRANSFER NFTs
- ADD NEW COLLECTIONS
- REQUEST SWAP
- SUPPORT SWAP NFT TO OTHER 
- SEARCH AND FIND NFT 

## RUN IN LOCALHOST
```
npm install
npm run build 
```
